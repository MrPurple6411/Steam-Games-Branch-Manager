# Steam Games Branch Manager

A Windows desktop application for managing different versions/branches of your Steam games. This tool allows you to create, switch between, and manage different versions of your Steam games.

## Features

- Create branches of Steam games
- Switch between different branches
- Rename existing branches
- Delete branches when no longer needed
- Manage multiple games in one interface

## Requirements

- Windows operating system
- .NET Framework 4.7.2
- Administrator privileges (required for creating symbolic links)
- Steam games installed

## Development

### Prerequisites

- Visual Studio Code
- .NET Framework 4.7.2 SDK
- Recommended VS Code extensions (will be prompted to install when you open the project)

### Building

1. Clone the repository
2. Open the project in VS Code
3. Run the `restore` task to restore NuGet packages
4. Run the `build-debug` task to build the project

### Debugging

1. Press F5 to start debugging
2. Select the "Debug" configuration for development or "Release" for testing the release build
